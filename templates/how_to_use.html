How to use



You can choose an image of your choice to encode the message and upload it. You have an option of using the default key or using a custom key to encrypt the message. If you have used a custom key, then remember to notify the recipient of the key. Text can be decrypted only if the recipient enters the custom key.

While using a custom key, it is better to use a key of more than 10 unique characters as it helps in creating a more secure encrypted text. After encoding, the image will be available to download only in the ".png" format as it uses a loseless data compression algorithm because of which all the data will be intact within the image.
